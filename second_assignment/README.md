This code involves:

- Histogram calculation;
- Linear transformations;
- Image equalization;
- Histogram “matching”;
- Convolution;
- Zoom in and zoom out;
- Negative images;
- Contrast factor;
- Image brightness factor;
- Image rotation;
- Filtering in the spatial domain.

  
The filters used were:

- Gaussian; 
- Laplaciano;
- Passa Altas;
- Prewitt Hx;
- Prewitt Hy;
- Sobel Hx;
- Sobel Hy.
